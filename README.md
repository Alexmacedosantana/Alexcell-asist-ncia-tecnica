📱 AlexCell Assistência Técnica
Projeto desenvolvido para a AlexCell Assistência Técnica, especializada em manutenção de smartphones e venda de acessórios em Rio Branco - AC.

🚀 Sobre o Projeto
A AlexCell oferece serviços especializados para diversas marcas de celulares, com foco em qualidade, agilidade e atendimento ao cliente.

Este projeto foi desenvolvido para proporcionar uma presença digital moderna e facilitar o contato com os clientes.
